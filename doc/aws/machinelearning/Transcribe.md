## Amazon Transcribe
### 1. Amazon Transcribe
- Automatically convert speech to text  
→ 자동으로 음성을 텍스트로 변환시킨다.

- Uses a deep learning process called automatic speech recognition (ASR) to convert speech to text quickly and accurately  
→ 자동 음성 인식이라는 딥러닝 프로세스를 사용하여 음성을 텍스트로 매우 빠르고 정확하게 변환한다.

- Automatically remove Personally Identifiable Information (PII) using Redaction  
→ 몇 가지 기능을 알아둬야 한다. 첫 번째는 `Redaction`을 사용하여 `개인 식별 정보, PII`를 자동으로 제거할 수 있다는 것이다. 예를 들어 누군가의 나이, 이름, 사회보장번호 등이 있다면 자동으로 제거된다.

- Supports Automatic Language Identification for multi-lingual audio  
→ 다국어 오디오의 언어 식별이 가능하다. 프랑스어, 영어, 스페인어가 섞여 있어도 모두 인식한다.

- Use cases:
~~~
- transcribe customer service calls
→ 고객 대응 전화의 기록

- automate closed captioning and subtitling
→ 자동으로 자막 생성

- generate metadata for media assets to create a fully searchable archive
→ 완전히 검색 가능한 아카이브를 만들기 위한 미디어 자산에 대한 메타데이터 생성
~~~
